[菜單]developer開發機 ubuntu 18.04 x ASRock Deskmini B360

Wed, Mar 18, 2020
  • 0. Intro
  • 1. 需求
  • 2. Final Solution

0. Intro

在前一陣子手邊的Mac mini逐漸凋零之後,一直在想說要來弄一台電腦來代替它，東看西看考慮了好多

  1. New Mac mini (9th/10th) 加上ram+ssd，價格….
  2. DIY PC 鎖定mini-itx為主的配套，不過因為OS打算使用Ubuntu 18.04，所以要考慮主機板的driver問題… 有在網路上看到不少diy後在情的討論一直怕怕的… Asus目前官方有更新支援linux主機板的文件到2018/10…
  3. Intel NUC 會考慮NUC最主要的原因是因為看到ubuntu官方上有支援清單,不過有滿多型號較舊＆＆在台灣買不到…

1. 需求

  1. Interl i7 (9th up) + 32G ram
  2. SSD可擴充
  3. Ubuntu 18.04 相容
  4. 體積不要太大
  5. 長時間開機所以希望低耗電量 每天固定時間跑一些爬蟲與cronjob，偶而會ssh回來寫寫code跑一些MBPR跑不動的玩意xd

2. Final Solution

最終方案

  1. ASRock Deskmini B360 NTD 9500
  2. intel i7 9700 NTD 10500
  3. Transcend 16GB TSRam DDR4 2666 (TS2GSH64V6B) NTD 2500 *2 這邊要特別注意，在ASRock Deskmini B360的支援清單李並沒有這型號，不過Transcend這邊是可以看到支援B360的
  4. Intel 660P系列 2TB M.2 2280 PCI-E NTD 7000

最後的總價落在NTD 30000+整體上決的滿不錯的，加上安裝ubuntu18.04到目前也已經上工了約一個月，目前幾乎沒有遇到什麼問題，如果你也有類似的需求可以考慮看看這樣的菜單:)
