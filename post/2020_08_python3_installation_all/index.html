macOS設定Python3.8-VSCode開發環境

Thu, Aug 27, 2020

1. macOS 上安裝 Python3.8

如果你的macOS已經是最新版本的話，預設下應該已經安裝好Python3.8
如果沒有的話，我們可以透過Brew來安裝

brew install python3
brew upgrade python@3.8
brew install pipenv

安裝完成後可以確認一下目前的版本

python3 --version
#Python 3.8.5
python --version
#Python 2.7.16

如果你確定自己的環境中已經不會有python2的legacy code了
可以簡單編輯~/.zshrc的把python的command alias掉

# set pip3 and python3 as default.
alias python=python3
alias pip=pip3

再確認一次python version應該可以得到Python 3.8.5等資訊，到這邊就可以確認已經安裝完成。

2. 設定VSCode Extension for python3

VSCode的下載與安裝就不多說明了，接下來推薦幾個好用的Vscode extension for python開發

2.1 Python

安裝連結

如果你討厭裝很多extension，可以只裝這個就好，基本上linting, code, debugging, jupyter, auto-completion等常用開發所需都包含在內了。

2.2 Visual Studio IntelliCode

安裝連結

這是MS AI Powered的輔助extension，不只是給python開發用，也支持滿多其他語言，前陣子試用過後真心決的不錯。

2.3 Django

安裝連結

Django的extension，已經built-in了許多的snippet

2.4 Python Docstring Generator

安裝連結 個人滿推薦用來寫python code docs 的extension

2.5 Python Test Explorer for Visual Studio Code

安裝連結 Python Unittest 首選推薦extension。

2.6 Robot Framework Intellisense

安裝連結 Python自動化與測試的首選Robot Framework專屬的extension
