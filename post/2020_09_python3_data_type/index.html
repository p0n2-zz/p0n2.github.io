2020_09_python3_data_type

Tue, Sep 8, 2020

0. prerequisite

如果開發環境中還沒有安裝python3的，可以先參考之前整理的這一篇教學。

1. Numbers

在python中會遇見的number類型會有兩種型態，一個是整數型態的數值(ex: 2020)，另一種則是浮點數類型的值(ex: 3.1415)，以下分別來說明。

1.1 Integers

無論是正負值或是0，在python的定義裡都是屬於integer的資料型態。 以下為簡單的範例

p0n2_int = -10
print(p0n2_int)
# output -10

calc_int = 168 - 100
print(calc_int)
# output 100

後面我們會介紹更多data type，也會同時介紹integer與其他data type的convert。

1.2 Floating-Point Numbers

對於python的floating point類型，可以簡單理解成上面我們提過的integer data type 再加上小數點(decimal point)，像是10.0 或是3.14這兩個數值都可以歸類到float的資料類型。

p0n2_float = 16.8
print(p0n2_float)
# output 16.8

calc_float = 16.8 - 1.68
print(calc_float)
# output 15.12

1.3 Operators

由於前面提到的都是數值型的data type，因此我們要先提一下在python中的operators，operators主要是用來計算各種數值的加減乘除等，常見的operators如下。

x = 10
y = 1.68

print(x + y)
# output 11.68
print(x - y)
# output 8.32
print(x * y)
# output 16.8
print(x / y)
# output 5.9523809523809526
print(x // y)
# output 5.0
#整數除法
print(x % y)
# output 1.6000000000000003
# 取餘數
print(x ** y)
# output 47.86300923226383
# 指數運算

1.4 Converting Integers to Floats

python 提供了float()來幫忙轉換integer成floating number

p0n2_int = 10
print(float(p0n2_int))
# output 10.0

1.5 Converting Floats to Integers

python也提供了int()來幫忙轉換floating number成integer，會自動幫忙把小數點後面的數值無條件捨去，即便是1.99也是會被轉成1。

p0n2_int = 1.68
print(int(p0n2_int))
# output 1

1.6 透過operator來將integer轉成float

python3的特異功能～

p0n2_int = 10
print(type(p0n2_int))
# output <class 'int'>
print(p0n2_int/3)
# output 3.3333333333333335
print(type(p0n2_int/3))
# output <class 'float'>

2. Booleans

先來個簡單的範例，下面再接著介紹一些常跟boolean搭配的operators。

p0n2_bool = 2 > 1
print(p0n2_bool)
# output True

2.1 Comparison Operators

x = 10
y = 1.68

print(x == y)
# output False
# python是用==來判斷，有的語言會是===
print(x != y)
# output True
print(x < y)
# output False
print(x > y)
# output True
print(x <= y)
# output False
print(x >= y)
# output True

3. Strings

python中並沒有特別限定string要用單引號(')或是雙引號(")，不過因為在python中也滿常會用到多行的string，所以我自己是習慣都用雙引號來宣告string，也比較能保持一致性。

str = "Hi, p0n2"
print(str)
# output: Hi, p0n2

article = """\
hi, \
hello, \
p0n2
"""
print(article)
#output hi, hello, p0n2

python對於string可以應用的function太多了，後續我們再用另外的篇幅來討論。

3.1 Convert Numbers to Strings

python提供了str()可以幫忙將integer或是float轉成string，下面我們用範例來demo。

age = 99.9
print("p0n2 is " + str(age) + " years old")
# Output p0n2 is 99.9 years old

如果有興趣可以將str()移除看看會發生什麼error。

3.2 Convert Strings to Numbers

就像我們前面提過的integer與float互相轉換data type一樣，我們可以用float()與int()來將字串轉換成這兩種資料型態。 不過使用上仍須注意一下原先字串內的值，如果原先資串內容是浮點數但是convert時用int()處理，可能就會遇到error

float_str = "16.8"
int_str = "10"
print(float_str - int_str)
# output TypeError: unsupported operand type(s) for -: 'str' and 'str'
print(float(float_str) - int(int_str))
# output 6.800000000000001

print(int(float_str))
# output ValueError: invalid literal for int() with base 10: '16.8'

4. Lists

python中的lists是個人覺得好用的data type，基本上你可以任合你想得到的data item或是data type都集中在一個lists中，並用中括號[]宣告。

str_list = ["this", "is", "p0n2's", "blog"]
print(str_list[1])
# output is

int_list= [-2, -1, 0, 1, 2 ]
print(int_list[1])
# output -1

float_list = [3.14, 3.15, 3.16]
print(float_list[1])
# output 3.15

mixed_list = ["p0n2", -999, 3.14]
print(mixed_list[1])
# output -999

5. Tuples

相對於List，tuples是一中不能修改的list，通常會使用括號()來宣告。特性上也都類似，並不限制在tuples中的item data type。

mixed_tuple = ["p0n2", -999, 3.14]
print(mixed_tuple[1])
# output -999

5.1 Convert to Tuples and Lists

前面有介紹了Tuples跟Lists，如同我們有介紹過的兩者有mutable/immutable的差異，因此在實際的application中，也常面臨到需要互相convert的狀況。 在python中提供了tuple()與list()這兩個functio來協助convert的工作，我們用範例來demo

mixed_list = ["p0n2", -999, 3.14]
print(tuple(mixed_list))
# output ('p0n2', -999, 3.14)

mixed_tuple = ["p0n2", -999, 3.14]
print(list(mixed_tuple))
# output ['p0n2', -999, 3.14]

另外，list()與tuple()這兩個func也常被用來轉換字串。

print(list("p0n2"))
# output ['p', '0', 'n', '2']
print(tuple("p0n2"))
# output ('p', '0', 'n', '2')

6. Dictionaries

Python中Dictionaries也是食物中滿常用的一種data type，也常常被簡寫成dict，在python中通常會透過大括號{}來宣告。

與前面介紹的list/tuple差異最大的地方為，在list/tuple中如果我們想要取的item值，通常只能透過key值來取的，而dict最大的差異就是可以自定義key value，並且可以隨時透過想要的key值來快速的取得資料。


p0n2_dict = {'name': 'p0n2', 'age': 99, 'sex': 'unknown', 'country': 'Taiwan'}
print(p0n2_dict)
# output {'name': 'p0n2', 'age': 99, 'sex': 'unknown', 'country': 'Taiwan'}
print(p0n2_dict["country"])
# output Taiwan
